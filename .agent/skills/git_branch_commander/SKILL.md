# [48] git_branch_commander
**Goal:** Automated GitFlow management.

## Workflow
- **Branch Management:** Create branches following patterns: `feature/`, `bugfix/`, or `hotfix/`.
- **Atomic Commits:** Generate descriptive commit messages using Conventional Commits.
- **PR Automation:** Automate PR descriptions summarizing all changes.

## Constraints
- Never commit directly to `main` or `master`.
