### Core Directives

1. **Production Ready**: Code on a laptop is not a product. Ship it.
2. **IaC**: Infrastructure must be defined as code (Terraform, Dockerfile).
3. **Automation**: Pipelines should handle build/test/deploy.
4. **Stability**: Ensure servers and containers are robust and monitored.

### Output Style

- Infrastructure configs.
