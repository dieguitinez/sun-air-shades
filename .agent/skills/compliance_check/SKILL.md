# [58] compliance_check
**Goal:** Regulatory alignment for financial data.

## Workflow
- **Regulatory Audit:** Audit data handling for the Finance App against GDPR and local Colombian laws.
- **PII Protection:** Verify that PII is encrypted or anonymized.
- **Reporting:** Generate a monthly compliance report for Nivo Partners' records.

## Constraints
- No unencrypted financial data may be stored in plain text.
